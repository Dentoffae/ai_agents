# 🤖 AI Agent — Telegram Bot

Локальный AI-агент на базе **OpenAI GPT**, работающий через **Telegram**.  
Агент понимает запросы на естественном языке, самостоятельно выбирает нужный инструмент и возвращает структурированный ответ.

---

## ⚙️ Установка и настройка

### 1. Требования

- Python **3.10+**
- Аккаунт [OpenAI](https://platform.openai.com/) с API-ключом
- Telegram-бот, созданный через [@BotFather](https://t.me/BotFather)

### 2. Установка зависимостей

```bash
cd AI_AGENT
pip install -r requirements.txt
```

### 3. Получение API-ключей

**OpenAI API Key:**
1. Зайди на [platform.openai.com/api-keys](https://platform.openai.com/api-keys)
2. Нажми **Create new secret key**
3. Скопируй ключ вида `sk-proj-...`

**Telegram Bot Token:**
1. Открой Telegram, найди [@BotFather](https://t.me/BotFather)
2. Напиши `/newbot`
3. Введи имя бота (например: `My AI Assistant`)
4. Введи username (например: `my_ai_assistant_bot`) — обязательно заканчивается на `bot`
5. BotFather выдаст токен вида `7123456789:AAFxxx...`

### 4. Настройка `.env`

Открой файл `agent/.env` и заполни:

```env
OPENAI_API_KEY=sk-proj-ваш-ключ-openai
TELEGRAM_BOT_TOKEN=7123456789:AAFваш-токен-бота
```

### 5. Запуск бота

```bash
cd agent
python bot.py
```

При успешном запуске в терминале появится:
```
✅ Бот @your_bot_name запущен. Нажми Ctrl+C для остановки.
```

## 🛠 Инструменты агента

Агент автоматически выбирает нужный инструмент исходя из запроса пользователя.

---

### 🌐 Поиск в интернете

Источник: **DuckDuckGo**

**Примеры запросов:**
```
Найди последние новости про ChatGPT
Что такое квантовые компьютеры?
Кто такой Илон Маск?
```

---

### 🌤 Погода

Источник: **Open-Meteo API** (бесплатно, без ключа)  
Геокодинг: **Open-Meteo Geocoding API**

Возвращает: температуру, скорость ветра, условия погоды, влажность.

**Примеры запросов:**
```
Какая погода в Москве?
Погода в Токио сейчас
Как погода в Дубае?
```

---

### 💰 Курс криптовалют

Источник: **CoinGecko API** (бесплатно, без ключа)

Поддерживаемые монеты: `bitcoin`, `ethereum`, `solana`, `dogecoin`, `litecoin`, `ripple`, `cardano`, `binancecoin` и другие.  
Поддерживаемые валюты: `usd`, `eur`, `rub` и другие.

**Примеры запросов:**
```
Сколько стоит Bitcoin в рублях?
Курс Ethereum в долларах
Цена Solana
```

---

### 💱 Курс обычных валют

Источник: **ExchangeRate.host** + **Open.er-api.com** (резервный)

Поддерживаемые валюты: USD, EUR, RUB, AED, BYN, GBP, JPY, CNY, UAH, KZT, TRY, CHF и другие.

**Примеры запросов:**
```
Сколько стоит доллар в рублях?
Курс евро к дирхамам
USD → BYN
Курс фунта к евро
```

### 📋 Напоминания

Хранилище: `agent/reminders.json`

Агент понимает даты на естественном языке и конвертирует их в формат `YYYY-MM-DD HH:MM`.

**Операции:**
- Добавить напоминание
- Просмотреть все напоминания
- Удалить напоминание по ID

**Примеры запросов:**
```
Напомни завтра позвонить клиенту в 14:00
Поставь напоминание на 25 марта в 10:00 — встреча с врачом
Покажи мои напоминания
Удали напоминание 3
```

**Формат хранения (`reminders.json`):**
```json
[
  {
    "id": 1,
    "text": "Позвонить клиенту",
    "datetime": "2026-03-24 14:00",
    "created_at": "2026-03-23 10:15"
  }
]
```

### 🧮 Калькулятор

Реализация: безопасный `eval` с пространством имён модуля `math`.

**Поддерживаемые операции:**

| Категория | Примеры |
|-----------|---------|
| Арифметика | `+`, `-`, `*`, `/`, `**` (степень), `%` |
| Функции | `sqrt()`, `log()`, `log10()`, `exp()` |
| Тригонометрия | `sin()`, `cos()`, `tan()`, `asin()` |
| Константы | `pi`, `e` |
| Прочее | `abs()`, `round()`, `min()`, `max()` |

**Примеры запросов:**
```
Сколько будет (25 * 4) + 10^2?
sqrt(144) + log(100)
sin(30) * 2
2^10
pi * 5^2
```

## 🧠 Память и контекст

- История диалога сохраняется в `agent/memory.json`
- Хранятся последние **50 сообщений**
- В контекст каждого запроса передаются последние **20 сообщений**
- У каждого пользователя Telegram **отдельная история**
- Память сохраняется между перезапусками бота
- Очистить память: `/clear`

## 📊 Логирование

Система логирования пишет одновременно в два потока:

| Поток | Уровень | Что пишет |
|-------|---------|-----------|
| Консоль | `INFO` | Запросы, ответы, вызовы инструментов |
| `agent/agent.log` | `DEBUG` | Всё подробно: токены, аргументы, трейсбеки |

**Пример лога в консоли:**
```
[INFO] 14:23:01  🔧 get_weather(city='Москва')  [312ms]  → 🌤 Погода в Москва...
[INFO] 14:23:02  🤖 Ответ агента [1843ms, итераций: 2]
```

**Пример лога в файле:**
```
[DEBUG   ] 2026-03-23 14:23:01.042  Итерация 1 / 5
[DEBUG   ] 2026-03-23 14:23:01.356  LLM ответ [734ms] | tokens: prompt=512, completion=89, total=601
[INFO    ] 2026-03-23 14:23:01.780  🔧 get_weather(city='Москва')  [312ms]  → 🌤 ...
```

---

## 🔒 Безопасность

- **Терминальные команды:** заблокированы опасные операции (`rm -rf`, `format` и др.)
- **Калькулятор:** `eval` выполняется в изолированном пространстве имён без доступа к системе
- **API-ключи:** хранятся в `.env` файле, не попадают в код
- **HTTP-запросы:** таймаут 15 секунд, защита от зависания

## 🐛 Решение частых проблем

**`ModuleNotFoundError`**
```bash
pip install -r requirements.txt
```

**`OPENAI_API_KEY не найден`**
Проверь файл `agent/.env` — ключ должен быть без пробелов и кавычек:
```
OPENAI_API_KEY=sk-proj-...
```

**`TELEGRAM_BOT_TOKEN не найден`**
Аналогично — добавь токен в `agent/.env`:
```
TELEGRAM_BOT_TOKEN=7123456789:AAF...
```

**Бот не отвечает**
- Убедись что `python bot.py` запущен и не завершился с ошибкой
- Проверь `agent/agent.log` на наличие ошибок
- Убедись что OpenAI API ключ активен и есть баланс

**Ошибка `ImportError: cannot import name 'AgentExecutor'`**
Этот проект не использует LangChain-агентов. Убедись что запускаешь `bot.py`, а не старую версию файлов.
